# 00-Organisation

Dieser Ordner enthält **alle organisatorischen und administrativen Informationen** zum Projekt **CitBin**.
Hier werden zentrale Projektinformationen, Planungsdokumente, Protokolle und organisatorische Entscheidungen abgelegt.

Der Ordner dient als **gemeinsame Informationsbasis für alle Teams** und stellt sicher, dass Projektstruktur, Rollen, Termine und Entscheidungen für alle nachvollziehbar dokumentiert sind.
