# Unbegrifflichkeiten klären

- **FastAPI:**  
modernes, schnelles Webframework für Python -> ermöglicht schnell und effizient performante Wen-APIs zu erstellen

- **REST-API:**  
Schnittstelle, über die Prgramme mit dem Internet miteinander kommunizieren -> sie nutz HTTP-Anfragen und sendet im JSON-Format
 
- **PostgreSQL mit SOLModel:**  

  relationale Datenbank mit einer Python Bibliothek um Daten einfach als Objekte zu verwalten -> PostgreSQL speichert die Daten, und SQLModel erlaubt, diese Daten in Python wie normale Klassen zu definieren und zu bearbeiten, statt rohe SQL-Befehle zu schreiben.

 
- **HTTP Uplink vom ChirpStack:**  
Methode, die empfangene Sensordaten an externe Anwendungen sendet -> Sobald ein LoRaWAN-Gerät Daten sendet, leitet ChirpStack diese per HTTP (meist als JSON) an eine definierte URL (Webhook) weiter.

- **IoT-Gerät:**  
 ein physisches Gerät, welches mit dem Internet verbunden ist, es kann Daten senden und Empfangen -> Es sammelt Informationen (z. B. Temperatur, Standort) und kommuniziert diese automatisch über ein Netzwerk.
