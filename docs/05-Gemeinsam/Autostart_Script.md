# Autostart-Script
## 1. Vorraussetzungen
- Stelle sicher, dass das Projekt ohne Fehler manuell gestartet werden kann
- Sorge gegebenenfalls mit `chmod +x citbin-dev.sh` dafür, dass das Script ausführbar ist
- Ändere die `$CITBIN_PATH`-Variable im Script zum Pfad deines citbin-Verzeichnisses, ausgehend von deinem `home`-Verzeichnis (`~`)
    * z. B. `$HOME/citbin` oder `$HOME/Projects/citbin`
## 2. Starten
- Öffne ein neues Terminal-Fenster
- Navigiere zum citbin-Verzeichnis
- Führe das Script mit `./citbin-dev.sh` aus
## 3. Beenden
Schließe das Terminal-Fenster
