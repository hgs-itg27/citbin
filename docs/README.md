# Dokumente des Projekts

Hier liegen alle Unterlagen und Dateien des Projekts CitBin.

## Inhalte
- Jedes Team hat einen eigenen Ordner, in dem teamspezifische Informationen und Dateien gesammelt werden.
- Informationen, die für alle gelten, gehören in *05-Gemeinsam*.
- Offizielle Entwicklungsstände kommen nach *06-Abgabe*.

## Hauptordner
In jedem Hauptordner liegt eine *README.md* mit:
- Zweck des Ordners
- zuständiges Team
- was dort abgelegt werden soll

## Regeln für Dateinamen
Feste Benennungen:
- Protokolle: *YYYY-MM-DD_Titel.md*
- Sitzungen: *YYYY-MM-DD_Sitzung.md*
- Versionierte Dokumente: *Pflichtenheft_v1.md*, *Pflichtenheft_v2.md*
